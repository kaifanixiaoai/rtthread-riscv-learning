# Week 3 Demo Plan

## 项目目标
在 RT-Thread qemu-virt64-riscv BSP 中新增一个自定义 shell 命令模块，并在 QEMU 中验证命令可用。

## 计划实现的命令
1. hello_week3
2. show_build_info
3. mem_info_simple

## 预期成果
- 新增一个真实的 C 源码文件
- 能通过 RT-Thread BSP 编译
- 能在 QEMU 中运行
- 能在 msh 中调用并看到输出
- 形成日志、总结和验证记录

## 当前理由
这个项目比单纯修改打印更像真实实战，同时复杂度又适合当前阶段。
